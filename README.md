# zhuhuanhuan-pc-project

> pc端练习项目

## Build Setup

``` bash
# install dependencies
npm install

#先生成vendor文件
npm run dll

#本地server
npm run dev

# 线上环境构建
npm run build prod

# 沙箱环境构建
npm run build box

# 测试环境构建（如果build后不加参数默认为test）
npm run build test

# build for production and view the bundle analyzer report
npm run build --report
